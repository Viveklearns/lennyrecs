Option 4

The mobile design would stretch to fill the entire browser width on desktop.

Everything stays the same, just wider. Book covers, text, and spacing all scale up proportionally.

📱 Impact on Mobile View
Zero impact. All of these options use CSS media queries that only activate on screens ≥768px wide. Your phone will always show the current mobile design exactly as it is now — single column, full width, with all the same spacing and sizing. You'd only see these desktop layouts when viewing on a laptop or desktop computer.
